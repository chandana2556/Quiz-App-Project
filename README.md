# Quiz App

A full-stack Quiz Application built using Java Spring Boot, Spring Data JPA, Hibernate, MySQL, HTML, CSS and JavaScript.

## Features

- User Login
- Quiz Category Selection
- Dynamic Quiz Generation
- Previous & Next Navigation
- Quiz Submission
- Automatic Score Calculation
- REST API Integration
- MySQL Database
- Responsive UI
